When adding a package in ALCOM

  • This repository prioritizes the VCC redirect because most users install VPM packages through VRChat Creator Companion.
  • A reliable dual redirect for both VCC and ALCOM is not technically possible because both clients use the same vcc:// URL scheme.
  • To use ALCOM, add the repository URL manually from inside ALCOM.
  • You can also add the repository to VCC first. After it is added, ALCOM can read the same repository and use it when opening a Unity project.
