ALCOMでパッケージを追加する場合

  • VCCの利用人口が多いため、このリポジトリではVCCへのリダイレクトを優先しています。
  • 技術的に、VCCとALCOMの両方へ確実に分岐する複数のリダイレクトは組めません。どちらも同じvcc:// URLスキームを使うためです。
  • ALCOMで使う場合は、ALCOM内から次のリポジトリURLを手動で追加してください。
    https://nickel-jp.github.io/avatar-recovery-unity/index.json
  • 一旦VCCに追加しても問題ありません。追加後はALCOMにも反映されるため、Unityプロジェクトを立ち上げる時にALCOM側から利用できます。

Manual repository URL:
https://nickel-jp.github.io/avatar-recovery-unity/index.json
